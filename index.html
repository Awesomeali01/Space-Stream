How It Works - Guide

1. Download

Get the Space Stream app for your PC. Our installer makes setup a breeze.

2. Install Dependencies

Set up ADB and Scrcpy. Use 'choco install scrcpy' in PowerShell (admin mode) for easy installation.

3. Customize

Open Space Stream and tailor the settings to your preferences for optimal performance.

4. Connect

Use USB or wireless connection. For wireless, ensure both devices are on the same network and enable wireless debugging.

5. Find IP

Use the 'Find IP Address' button or check your phone's Wi-Fi settings for ip-address.

6. Start Streaming

Click 'Connect Device' and start controlling your Android device from your PC!
